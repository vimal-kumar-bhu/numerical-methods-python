# Interpolation Methods

Python implementation of interpolation techniques:

* Lagrange Interpolation
* Hermite Interpolation
* Spline Interpolation

## 📌 Use

* Estimate missing values between data points
* Smooth curves from discrete data
* Lagrange → general data points
* Hermite → value + derivative known
* Spline → smooth curve fitting

## 🌍 Real-World Applications

* 📊 Data Science → missing data filling & preprocessing
* 📈 Finance → stock trend approximation
* ⚙️ Engineering → curve design & simulations
* 🛰️ Physics → trajectory and motion approximation
* 🎮 Computer Graphics → smooth curves & animations

## 📂 Files

* `lagrange_interpolation.ipynb`
* `hermite_interpolation.ipynb`
* `spline.ipynb`
